#Sistema de Chamados Internos de TI — SM Obras

Sistema web completo para gerenciamento de chamados internos de TI, desenvolvido em Python + Flask, com foco em ambientes corporativos e órgãos públicos. O projeto oferece controle de tickets, SLA automático, painel administrativo, gestão de usuários, exportação de relatórios, comentários, anexos e notificações por e-mail.

🚀 Tecnologias utilizadas
Python
Flask
SQLite
HTML5 / CSS3
JavaScript
FPDF
SMTP (Notificações por E-mail)

✨ Funcionalidades
🔐 Sistema de login com níveis de acesso (Usuário, TI e Administrador)
📋 Abertura e gerenciamento de chamados
⏱️ SLA automático por prioridade
📎 Upload de anexos
💬 Comentários e histórico completo do chamado
📊 Dashboard com gráficos mensais
📄 Exportação em CSV e PDF
📧 Notificações automáticas por e-mail
👨‍💼 Painel administrativo para criação e gerenciamento de usuários
🌙 Interface moderna com tema institucional dark

🎯 Objetivo do projeto

Criar uma plataforma interna moderna e eficiente para controle de suporte técnico, facilitando o atendimento entre colaboradores e equipe de TI, além de melhorar organização, rastreabilidade e tempo de resposta.
